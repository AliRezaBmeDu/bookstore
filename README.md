# ğŸ“– Bookstore <a name="about-project"></a>

**Bookstore** aims to create a book store web application, which delivers a boooks section as well as a category section. In this application, users can navigate through the collection of books. They can add new book and delete any book in real-time. In the category page, users have the filter feature to see the specific collection of books. The project is built with react framework.
